Movie API

A REST API for scraping movie and TV show data, deployable to Google Cloud Run.

Cloud Run Deployment

This API is containerized and ready for Google Cloud Run.

# 1. Build and push image
gcloud builds submit --tag gcr.io/YOUR_PROJECT/movie-api

# 2. Deploy to Cloud Run
gcloud run deploy movie-api \
  --image gcr.io/YOUR_PROJECT/movie-api \
  --platform managed \
  --allow-unauthenticated

API Endpoints

GET /health

Health check endpoint. Returns {"status": "ok"}.

GET /providers

List available providers (flixhq, sflix, etc).

GET /search?q={query}&provider={name}

Search for movies and TV shows.

GET /stream?serverId={id}&provider={name}

Get decrypted stream URL (m3u8) for playback.
