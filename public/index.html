Movie API

A REST API for scraping movie and TV show data, deployable to Google Cloud Run.

Cloud Run Deployment

This API is containerized and ready for Google Cloud Run.

# 1. Build and push image
gcloud builds submit --tag gcr.io/YOUR_PROJECT/movie-api

# 2. Deploy to Cloud Run
gcloud run deploy movie-api \
  --image gcr.io/YOUR_PROJECT/movie-api \
  --platform managed \
  --allow-unauthenticated

API Endpoints

API Endpoints

GET /home?provider={name}

Get home page content including Trending, Latest Movies/TV, Latest Anime, and Coming Soon.

{
  "trending": [{ "title": "...", "url": "...", "type": "movie", "poster": "..." }],
  "latest_movies": [...],
  "latest_tv_shows": [...],
  "anime": [...],
  "coming_soon": [...]
}
GET /search?q={query}&provider={name}

Search for movies and TV shows.

[
  { "title": "Avatar", "url": "...", "type": "movie", "poster": "..." }
]
GET /mediaId?url={url}&provider={name}

Extract the unique Media ID from a provider URL.

{ "media_id": "123456" }
GET /seasons?mediaId={id}&provider={name}

Get seasons for a TV show. (Returns empty for movies)

[{ "id": "...", "name": "Season 1" }]
GET /episodes?id={id}&isSeason={bool}&provider={name}

Get episodes for a specific season (TV) or movie.

Note for FlixHQ Movies: When isSeason=false, this endpoint returns the list of available Servers (e.g., UpCloud, Vidcloud) directly as "episodes". You should use these IDs directly with the /link endpoint.

[{ "id": "...", "name": "Episode 1" (or "UpCloud" for movies) }]
GET /servers?episodeId={id}&provider={name}

Get available servers for a TV show episode.

Note: For FlixHQ movies, skip this step and use the IDs from /episodes directly.

[{ "id": "...", "name": "VidCloud" }]
GET /link?serverId={id}&provider={name}

Get the embed link for a specific server.

{ "link": "https://rabstream.net/embed..." }
GET /stream?serverId={id}&provider={name}

Get decrypted stream URL (m3u8) for playback (if supported).
